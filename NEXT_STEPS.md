# Future Roadmap: Khitab Evolution

While the core digital heirloom experience is fully functional, the following features represent the next horizon for Khitab's growth.

## 1. Real-Time Tracking (WebSockets)
- **Status**: Backend ready, Frontend polling 30s.
- **Goal**: Implement full WebSocket events to push "Letter Arrived" and "Letter Dispatched" notifications instantly without refreshing the dashboard.

## 2. Reverse Geocoding Integration
- **Status**: Placeholder City/Country used.
- **Goal**: Integrate a service to automatically resolve user coordinates during signup into human-readable locations.

## 3. Advanced Pen-Pal Matching
- **Status**: Basic intersection scoring.
- **Goal**: Implement a more sophisticated recommendation engine that considers geographic rarity and sentiment matching.

## 4. Stationery Customization
- **Status**: Base "Digital Heirloom" paper.
- **Goal**: Allow "Premium Study" members to choose custom paper textures and upload custom Wax Seal designs.

## 5. Mobile Companion App
- **Status**: Desktop/Web responsive.
- **Goal**: Build a native companion for receiving haptic notifications to heighten the experience.
